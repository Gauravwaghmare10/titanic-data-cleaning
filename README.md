# 🧹 Titanic Dataset - Data Cleaning & Preprocessing (Google Colab Ready)

This repository contains a Google Colab-ready Jupyter notebook that demonstrates step-by-step data cleaning, preprocessing, and outlier removal on the classic Titanic dataset.

## 📌 Project Objective
Prepare raw Titanic passenger data for machine learning by performing:
- Handling missing values
- Encoding categorical variables
- Feature scaling (standardization)
- Outlier detection and removal
- Exporting cleaned data
